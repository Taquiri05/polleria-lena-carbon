# Sistema Integral de Gestión para Pollería Leña y Carbón

Sistema web desarrollado con metodología SDD (Spec-Driven Development) y Scrum para el curso IS-489 Pruebas y Aseguramiento de Calidad — UNSCH 2026.

## Stack Tecnológico
- **Backend:** Python + Flask + SQLAlchemy + JWT
- **Frontend:** React + Vite + Tailwind CSS + shadcn/ui
- **Base de datos:** MySQL 8.0
- **Pruebas:** pytest (90.71% cobertura)

2. Carta Digital
3. Takeaway
4. Panel de Administración
5. Usuarios y Roles
